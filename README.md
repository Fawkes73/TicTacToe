
# Tic Tac Toe

## Overview

The Tic Tac Toe game in React.

Stack:

- React
- Styles with Sass



## Develop locally


```bash
1. Clone this repo by running `git clone https://github.com/Fawkes73/TicTacToe.git`
```

```bash
2. Install dependencies by running `npm install`
```


```bash
3. Run `npm run start`
```
